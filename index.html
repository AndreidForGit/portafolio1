MiniGPT

MiniGPT, es un bot para Telegram escrito en JavaScript utilizando las librerías de Telegraf y OpenAI. Este bot hace uso de las APIs de Telegram, OpenAI y Writesonic, lo que te permitirá disfrutar de una experiencia de chat similar a la de ChatGPT dentro de la aplicación de Telegram

Codigo en Github

CryptoPricy

CryptoPricy, es un bot para Telegram escrito en Python utilizando las librerías de Telegram, CoinGecko y Pickle. Este bot hace uso de las APIs de Telegram y CoinGecko, lo que permite consultar el precio de todas las cryptomonedas en la plataforma de CoinGecko
